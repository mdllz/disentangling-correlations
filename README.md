This repository contains the analysis code for the manuscript "Disentangling Within- and Between-Subject Correlations in Cognitive Models: The Essential Role of Hierarchical Estimation" submitted by Michelle Donzallaz, Niek Stevenson, Andrew Heathcote, and Dora Matzke.

It is linked to the following OSF repository: https://osf.io/z5nqm/
The simulated data with 10000 trials per condition as well as some of the fit objects (hierarchical estimation on simulated data) can be found there.
